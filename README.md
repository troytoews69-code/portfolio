# Troy Lussier - Full Stack Developer Portfolio

## 🚀 About This Portfolio

A modern, responsive, and interactive portfolio website showcasing my journey from 20+ years of carpentry craftsmanship to full-stack web development. This portfolio demonstrates my passion for creating innovative web solutions and bringing ideas to life through clean, efficient code.

### Portfolio Highlights

- **Interactive Weather Dashboard** - Real-time weather data with 3-day forecasts and location search
- **Quiz Application** - Dynamic trivia game with multiple categories, difficulty levels, and scoring
- **Task Manager** - Full-featured productivity app with priority levels and local storage
- **Scientific Calculator** - Advanced calculator with history and theme switching
- **GitHub Integration** - Live GitHub activity feed and repository stats
- **Responsive Design** - Fully responsive across all devices and screen sizes

## 🛠️ Built With

This portfolio is built using modern web technologies:

- **HTML5** - Semantic markup and accessibility
- **CSS3** - Advanced styling with Flexbox, Grid, and animations
- **JavaScript (ES6+)** - Interactive features and API integrations
- **Font Awesome** - Icon library for enhanced UI
- **WeatherAPI** - Real-time weather data integration
- **Open Trivia DB** - Quiz questions API
- **GitHub API** - Repository and activity data
