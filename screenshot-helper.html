How to Take Screenshots

  1. Click on each project link below to open it in a new tab
  2. Let the project load completely (wait for API data if applicable)
  3. Use Windows Snipping Tool (Win + Shift + S) or your browser's screenshot feature
  4. Save with the exact filename shown (e.g., "weather-app-screenshot.jpg")
  5. Recommended size: 800x500 pixels (16:10 ratio)
  6. Place all screenshots in your portfolio root folder

💡 Pro Screenshot Tips

  • Take screenshots in a clean browser window (no bookmarks bar)
  • Zoom to 100% for best quality
  • For Weather App: Wait for weather data to load before screenshotting
  • For Quiz App: You can screenshot the start screen or an actual question
  • For Task Manager: Add 1-2 sample tasks first for a better screenshot
  • For Calculator: Consider showing a calculation in progress
  • Use a consistent background/theme across all screenshots
  • Compress images after saving (use tinypng.com or similar)
