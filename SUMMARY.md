## 🎯 URL Shortening Guide

### Current URL (Typical):
```
https://troytoews69-code.github.io/portfolio-project-part-2-more-fuctionallity/
```

### Option 1: Rename Repository (Easy)
**New URL Example:**
```
https://troytoews69-code.github.io/portfolio/
```
**Steps:**
1. Go to GitHub repository
2. Settings → Repository name
3. Change to "portfolio" or "troy-lussier"
4. Click Rename

### Option 2: User Page (Best for GitHub)
**New URL:**
```
https://troytoews69-code.github.io/
```
**Steps:**
1. Create new repository named: `troytoews69-code.github.io`
2. Move portfolio files there
3. This becomes your main GitHub Pages site

### Option 3: Custom Domain (Most Professional)
**New URL Example:**
```
https://troylussier.dev
```
**Steps:**
1. Buy domain ($10-15/year)
2. Add to GitHub Pages settings
3. Update DNS records
4. Professional and memorable!
