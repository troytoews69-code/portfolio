# Resume Instructions

## 📄 Adding Your Resume

To enable the resume download feature:

1. **Create or Export Your Resume**
   - Export your resume as a PDF file
   - Recommended tools:
     - Microsoft Word → Save As → PDF
     - Google Docs → File → Download → PDF
     - Canva (for designed resumes)
     - Resume builders like Resume.io, Zety, etc.

2. **Name the File**
   - The file must be named **exactly**: `resume.pdf`
   - Case matters on some systems

3. **Place in Root Folder**
   - Put `resume.pdf` in the same folder as `index.html`
   - Path should be: `portfolio-project-part-2-more-fuctionallity/resume.pdf`

4. **Test the Download**
   - Open your portfolio in a browser
   - Click the "Download Resume" button in the hero section
   - Verify the PDF downloads correctly

## 📝 Resume Tips

Your resume should include:
- **Contact Information**: Email, phone, LinkedIn, GitHub
- **Professional Summary**: Brief overview highlighting your unique background
- **Technical Skills**: HTML, CSS, JavaScript, Git, etc.
- **Projects**: Link to your portfolio and highlight your best projects
- **Experience**: Emphasize your carpentry background and transition to tech
- **Education/Certifications**: Any coding bootcamps, courses, or certifications

## 🎯 Unique Selling Points to Highlight

1. **20+ Years Carpentry Experience**
   - Problem-solving skills
   - Attention to detail
   - Project management

2. **Career Transition Story**
   - Demonstrates adaptability
   - Shows passion and commitment
   - Fresh perspective on web development

3. **Hands-On Projects**
   - Weather Dashboard with API integration
   - Interactive Quiz Application
   - Task Manager with LocalStorage
   - Scientific Calculator

## 📌 Alternative: Link to Online Resume

If you prefer not to have a downloadable PDF, you can:
1. Create an online resume (LinkedIn, Indeed, etc.)
2. Update the button link to point to your online resume
3. Change the button text from "Download Resume" to "View Resume"
